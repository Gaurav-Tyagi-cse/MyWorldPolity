How To Prepare Yourself for Assam Direct Recruitment Grade-3 Computer Proficiency Test (CPT)


To prepare for the Assam Direct Recruitment Grade-3 Computer Proficiency Test (CPT), you need to focus on both theoretical knowledge and hands-on practical skills related to computers. The CPT is designed to assess your proficiency in using essential computer applications and understanding basic concepts. Here's a detailed guide to help you prepare effectively:

  • 1. Understand the Syllabus and Exam Pattern Before starting your preparation, make sure to understand the exact syllabus and topics covered in the CPT. Common sections usually include: 1. MS Office Suite: (Word, Excel, PowerPoint) 2. Basic Computer Operations: Typing, file management, etc. 3. Internet & Email Handling 4. Operating Systems (Windows) 5. Computer Security & Troubleshooting 6. Focus on the detailed notification from the Assam State Level Recruitment Commission to ensure you know the specific requirements.


  • 2. Microsoft Office (MS Word, Excel, PowerPoint) These are commonly tested areas in computer proficiency tests. You need to be well-versed with the following: MS Word: 1. Formatting text (bold, italics, underline, font size, style, etc.) 2. Creating tables, inserting images, page setup 3. Working with headers/footers, page numbering, and paragraph alignment MS Excel: 1. Basic formulas (SUM, AVERAGE, MIN, MAX) 2. Working with rows, columns, sorting, filtering, and basic data analysis 3. Creating charts and graphs, using cell references and ranges 4. MS PowerPoint: 1. Creating presentations with slides 2. Inserting images, graphs, and text 3. Adding animations and transitions between slides 4. Using different slide layouts Tip: Practice regularly with real documents and spreadsheets to improve your speed and accuracy.
